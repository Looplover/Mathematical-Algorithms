### 🚀 Projects Overview

</div>

#### 📐 Linear & Integer Programming
* **Simplex & Dual Simplex Methods**: A Python script building out the classic Simplex and Dual Simplex algorithms from scratch to efficiently solve linear programming problems.
* **Gomory Cut Approach**: An algorithmic extension implemented in Python designed to enforce integer constraints, effectively solving integer linear programming (ILP) problems.

#### 📉 Differential Equation Solvers
* **IVP & BVP Solvers**: A dedicated folder containing Python implementations for solving various types of differential equations, specifically handling Initial Value Problems (IVPs) and Boundary Value Problems (BVPs).
* **Mathematical Visualization**: Includes plotting functionalities integrated with the numerical methods to graph the solutions, making it easier to analyze the accuracy and behavior of the underlying equations.

#### 🧮 Numerical Analysis
* **MATLAB Methods Suite**: A collection of MATLAB scripts executing various numerical analysis methods, ideal for testing algorithm stability and computational mathematics.
